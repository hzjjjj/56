------------功能-------------
输入数字1-8 输出乾坤

高岛起卦：
输入顺序3个数字：上卦 下卦 变卦的动爻
变爻序从下往上数  

输出：卦名卦象和其变卦 
输入一个（本）卦 输出其互错综三个卦

------------要解决问题-------------
1 输入只能是1-8数字，减少程序报错负担。
2 输入上下卦数，算法自动生成64卦本卦，并输出
3 自动掷卦 生成3个数字
4 修改起卦等待时间1-0.5
5 Nginx多站并联展示
